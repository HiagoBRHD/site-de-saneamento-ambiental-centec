Transformação de alto nível do guia de estudos de Saneamento Ambiental para uma plataforma interativa, modular e de alto desempenho.

## 🚀 Funcionalidades "Pro Max"

-   **🎨 Design Eco-Tech**: Interface premium com Glassmorphism, modo escuro e layout Bento Grid.
-   **📚 Biblioteca Integrada**: Acesso direto a 42 livros em PDF mapeados para cada disciplina.
-   **⏱️ Estudo Inteligente**: Ferramenta de Pomodoro e Flashcards (Active Recall) integradas.
-   **📊 Monitoramento de Progresso**: Visualização detalhada do avanço acadêmico por semestre.
-   **⚡ Performance de Elite**: Conteúdo inlined para carregamento instantâneo e suporte offline (PWA).
-   **📦 Portabilidade Total**: Build otimizado para Single File HTML — funciona em qualquer lugar.

## 🛠️ Tecnologias Utilizadas

-   **Frontend**: React 19 + Vite 7
-   **Estilização**: Tailwind CSS 4 (Aesthetics Pro)
-   **Ícones**: Lucide React
-   **Componentes**: Radix UI + Framer Motion (animações fluidas)
-   **Entrega de Arquivos**: Elite Blob-based PDF Delivery

## 📦 Como Rodar Localmente

1.  **Clone o repositório**:
    ```bash
    git clone https://github.com/seu-usuario/saneamento-2.0.git
    ```
2.  **Instale as dependências**:
    ```bash
    npm install
    ```
3.  **Inicie o servidor de desenvolvimento**:
    ```bash
    npm run dev
    ```

## 🚀 Build e Deploy

Para gerar a versão final otimizada em um único arquivo HTML:
```bash
npm run build
```
O resultado estará na pasta `dist/`. Basta subir o conteúdo dessa pasta para o GitHub Pages, Vercel ou Netlify.

### 📝 Estrutura do Projeto

-   `src/data/curriculum/`: Estrutura da grade e conteúdo embutido.

-   `public/pdf/`: Biblioteca de livros técnicos.
